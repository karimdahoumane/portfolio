https://karimdahoumane.github.io/portfolio

### Local demo
Clone the project
  ```bash
  git clone https://github.com/karimdahoumane/portfolio.git
  ```
Go to the project directory
  ```bash
  cd portfolio
  ```
Install dependencies
  ```bash
  npm install
  ```
Start the server
  ```bash
  npm start
  ```
